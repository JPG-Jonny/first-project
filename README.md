# DEMO
***A LIGHT-WEIGHT PERSONAL PROJECT DEVELOPED TO PRACTICE PYTHON FUNDAMENTALS BEFORE STARTING MY 1ST SEMESTER OF ENGINEERING.***
***PROJECT NAME: WALLET TRACKER***
**KEY FEATURES**
# LIVE BALANCE CALCULATOR: AUTOMATICALLY REFLECTS NET BALANCE, WHENEVER INCOME IS ADDED AND EXPENSES ARE SUBTRACTED.
# SMART CATEGORIZATION: GROUPS EXPENSES UNDER 3 DIFFERENT LABELS: NEEDS, WANTS AND SAVINGS.
# INPUT PROTECTION: EMPLOYS RUNTIME VERIFICATION TO PREVENT SOFTWARE CRASHES CAUSED BY TEXT SYMBOLS OR NEGATIVE NUMBERS INSIDE CURRENCY.
**TECH STACK AND CONCEPTS USED**
# LANGUAGE: PYTHON
# DATA STRUCTURES: UTILIZING FUNCTIONS, LOOPS, LISTS , CLASS AND OBJECTS 
# CONTROL FLOW: STRUCTURED USING ROBUST WHILE LOOP, 'IF-ELSE-IF' STATEMENTS
